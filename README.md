Infobip Notifier
================

Provides [Infobip](https://infobip.com) integration for Symfony Notifier.

DSN example
-----------

```
INFOBIP_DSN=infobip://AUTH_TOKEN@HOST?from=FROM
```

where:
 - `AUTH_TOKEN` is your Infobip auth token
 - `HOST` is your Infobip host
 - `FROM` is the sender
